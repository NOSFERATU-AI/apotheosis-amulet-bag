# Apotheosis Amulet Bag

Мод для Minecraft 1.21.1 NeoForge, добавляющий сумку амулетов для Apotheosis.

## Информация

- Minecraft: `1.21.1`
- NeoForge: `21.1.233`
- Mod ID: `apoth_talisman_bag`
- Item ID: `apoth_talisman_bag:talisman_bag`
- Автор: `NOSFERATU`

Команда выдачи предмета:

```mcfunction
/give @p apoth_talisman_bag:talisman_bag
```

## Что делает мод

- Добавляет предмет `Сумка амулетов`.
- Сумка открывается по ПКМ.
- Внутри 5 слотов под амулеты/чармы Apotheosis.
- Слоты открываются постепенно за уровни игрока:
  - 1 слот — 100 уровней
  - 2 слот — 200 уровней
  - 3 слот — 300 уровней
  - 4 слот — 400 уровней
  - 5 слот — 500 уровней
- Открытие следующего слота: `Shift + ПКМ` по сумке.
- Эффекты амулетов внутри открытых слотов продолжают работать.
- Закрытые слоты визуально заблокированы.

## Зависимости

- NeoForge 1.21.1
- Apotheosis 1.21.1

Исходники восстановлены под переданный готовый `.jar` без добавления более поздних исправлений.
